
* Install 
	* https://facebook.github.io/react-vr/docs/getting-started.html#content

	```
	npm install -g react-vr-cli
	react-vr init PolkadotConsensusVR
	cd PolkadotConsensusVR
	```

* Run
	
	```
	npm start
	```

	* Open http://localhost:8081/vr/index.html
